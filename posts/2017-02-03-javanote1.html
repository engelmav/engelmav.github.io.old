An HOF in Java?

Posted on February 3, 2017 by Vincent Engelmann

Java’s Collections library offers a sort method, which takes as a parameter a generic object, sort of emulating a higher order function. You can take an ArrayList of some object and sort it by some field in the object. There are a few caveats. One is that you can’t sort by an int field. Fortunately you can just take the ints and convert them to Doubles.

Collections.sort(userList, new Comparator<User>() {
    @Override
    public double compare(User u1, User u2) {
        return new Double(u2.getCalcdUptime())
            .compareTo(new Double(u1.getCalcdUptime()));
    }
});
